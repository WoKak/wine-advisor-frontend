# WineAdvisor - frontend

Frontend of application advising on wine purposes based on data provided by client

#### Technology stack:

* TypeScript 2.5.3
* Angular 5
* HTML
* CSS
* Bootstrap 4
* ng-bootstrap
* npm

#### To run this app:
1) Install node
2) Clone code
3) Install npm
4) Install angular-cli

then you can start application with:

```
ng serve
```
